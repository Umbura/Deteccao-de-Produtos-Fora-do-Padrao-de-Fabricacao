# Pandora: Detecção de Produtos Fora do Padrão de Fabricação
O Projeto Pandora é um estudo desenvolvido para fins acadêmicos e educacionais, com o propósito de reproduzir e aplicar a metodologia proposta no artigo
“PaDiM: a Patch Distribution Modeling Framework for Anomaly Detection and Localization”
(Thomas Defard, Aleksandr Setkov, Angélique Loesch e Romaric Audigier, Université Paris-Saclay, 2020).

O objetivo central deste trabalho é explorar e implementar a metodologia PaDiM no contexto de inspeção visual industrial, aplicando-a ao problema de detecção de produtos fora do padrão de fabricação.

A pesquisa se baseia no uso de modelagem estatística de distribuições de características visuais para identificar anomalias — defeitos, falhas estruturais ou irregularidades — em imagens industriais.

Para a experimentação e validação da metodologia, foi utilizado o dataset público MVTec Anomaly Detection (MVTec AD)
, amplamente reconhecido na área de computer vision como referência para avaliação de algoritmos de detecção de anomalias em ambientes de produção.

Este projeto não utiliza o código original do PaDiM, mas sim uma reimplementação independente baseada na descrição metodológica apresentada pelos autores no artigo original.

Todo o desenvolvimento foi realizado sem fins lucrativos, tendo como finalidade a compreensão prática e científica do método, bem como sua aplicação em um cenário realista de controle de qualidade automatizado.
